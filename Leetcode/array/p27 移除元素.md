### 快慢指针覆盖法
### 思路
- fast:负责遍历数组
- slow: 负责维护新数组

### slow永远指向下一个元素该放的位置

```java
public static int removeElement2(int[] arr, int val){  
    int slow=0;  
    for(int fast=0;fast<arr.length;fast++){  
        if(arr[fast]!=val){  
            arr[slow]=arr[fast];  
            slow++;  
        }  
    }  
    return slow;  
}
```
