#滑动窗口 
2025-05-11

**"子数组"——连续** 
## 算法思想->[[滑动窗口]] O(n log n)
- “维护一段连续区间”  
- 可以向右滑动 可以扩大 可以缩小
### 类似双指针 但更像窗口

## 具体思路

- right:负责扩大窗口
- left :负责缩小窗口
- 
```java
public static int minSubArrayLen(int target, int[] nums){  
    int left=0;  
    int sum=0;  
    int res=0;  
    for (int right = 0; right < nums.length; right++) {  
        sum+=nums[right];  
        while (sum>=target){  
            res=(res==0)?right-left+1:Math.min(res,right-left+1);  
            sum-=nums[left];  
            left++;  
        }  
    }  
    return res;  
  
  
}
```
